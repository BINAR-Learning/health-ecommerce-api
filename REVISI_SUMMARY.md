```
Kamu pasti sering buka Instagram, kan? Nah, waktu kamu scroll feed,

bekerja di belakang layar.

Backend adalah bagian dari aplikasi web yang berjalan di server
(komputer yang nyala 24/7 di internet). Kalau frontend itu yang
kamu lihat dan klik-klik (tampilan website), backend itu yang
ngatur semua logika dan data di belakangnya.
```

## Tips untuk Instruktur

### Mengajar Modul Ini:

1. **Start dengan Section GitHub Copilot**

   - Demo live bagaimana Copilot works
   - Show ketika Copilot helpful dan kapan perlu manual

2. **Follow Vibe Coding Sessions**

   - Jangan skip! Ini core dari modul
   - Code bareng peserta step-by-step
   - Encourage mereka try prompts sendiri

3. **Use Aiman & Aila as Teaching Tools**

   - Roleplay sebagai Aiman (backend) dan Aila (frontend)
   - Show collaboration mindset
   - Demonstrate real developer interactions

4. **Leverage Challenges**

   - Kasih waktu peserta kerjakan sendiri
   - Discuss solutions setelah mereka try
   - Encourage creative solutions

5. **Visual Learning**
   - Pastikan semua GIF dan images ready
   - Show live demo untuk complex concepts
   - Use whiteboard untuk explain architecture

### Common Pitfalls to Address:

- Peserta langsung accept semua Copilot suggestions tanpa pahami
- Skip error handling karena dianggap "nanti aja"
- Nggak test incremental, langsung bikin semua baru test
- Copy-paste code tanpa understand
- Forget to read error messages
